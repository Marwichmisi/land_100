Comment alimenter land_100

Le flux est volontairement simple pour aller vite et garder une structure propre dans le repo. Tu peux lier une landing locale ou une URL externe.

1

Creer un dossier dans `100/`

Exemple: `100/02-super-glass-card/` puis place le code dans `index.html`.

2

Ajouter une entree dans `landings.json`

Remplis `name`, `category`, `level`, `thumbnail`, `tech`, `tags` puis sauvegarde.

3

Choisir le type de lien

Locale: `folder: "02-mon-projet"`. Externe: `url: "https://ton-site.com"`.

4

Push sur GitHub

GitHub Pages rebuild automatiquement. land_100 liste la nouvelle landing tout de suite.
